# Quina Frontend

Este projeto é uma aplicação frontend desenvolvida com React e TypeScript para gerenciar jogadores da Quina, um jogo de loteria brasileiro. Ele oferece operações CRUD (Create, Read, Update, Delete) para jogadores, além de funcionalidades para consultar o último jogo realizado ou um jogo específico.

## Funcionalidades

- **CRUD de Jogadores**: Gerencie informações dos jogadores da Quina.
- **Consultas de Jogos**: Visualize detalhes do último jogo da Quina ou pesquise por um jogo específico.

## Tecnologias Utilizadas

- React 19.2.0
- TypeScript 5.9.3
- Vite
- React Router DOM 7.12.0
- ESLint
- E outras dependências conforme listado no package.json

## Como Começar

### Pré-requisitos

- Node.js (versão 18 ou superior)
- npm

### Instalação

1. Clone o repositório:
   ```
    git clone https://github.com/krosscaal/ifnet-quina-frontend
   cd quina-frontend
   ```

2. Instale as dependências:
   ```
   npm install
   ```

3. Execute o servidor de desenvolvimento:
   ```
   npm run dev
   ```

4. Abra seu navegador e navegue para `http://localhost:5173` (ou a porta exibida no terminal).
